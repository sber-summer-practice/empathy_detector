# Определитель эмпатийности ответов ИИ-агента
Ищет четыре признака «неэмпатийности» в датасете — приказной тон; отказ помочь; наезды, оскорбления; канцелярит — и рассчитывает на их основе уровень эмпатийности.

По результатам тестирования на 20% тренировочного датасета:

• **F1** — 0,97

• **accuracy** — 0,88
